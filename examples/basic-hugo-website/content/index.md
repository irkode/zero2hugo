We are up and running a website from scratch using Hugo.

## Directory structure

Just four files needed.

```text
/basic-metal-website
│   hugo.toml              <-- site configuration
│   _index.md
│
├───content
│       index.md           <-- homepage
│
└───layouts
    │   home.html          <-- homepage content
    │
    └───_default
            baseof.html    <-- global html skeleton
```
